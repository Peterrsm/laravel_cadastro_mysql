# laravel_cadastro_mysql
# Requisitos recomendados para rodar a aplicação:
-> XAMPP \ WAMPP
-> Laravel Framework 7.30.4
-> PHP 7.2

# Instruções
1. Inicie o Apache e o MySQL pelo XAMPP \ WAMPP
2. Cole a pasta dentro do diretório "C:\xampp\htdocs"
3. Para iniciar a aplicação inicie o Git Bash dentro da pasta do projeto e insira o comando "php artisan serve"
4. Agora que a aplicação iniciou na porta 8000 basta digitar no navegador "localhost:8000"

Obs: Para rodar as migrations sega as instruções contidas no link a seguir: https://laravel.com/docs/8.x/migrations#running-migrations
